This project uses **PostgreSQL** managed via Docker. Follow these steps to get your local environment running.

### 1. Start the Containers
Run the following command in your terminal at the root of the project:
```bash
docker-compose up -d
2. Stop the Containers
To stop the database without deleting data:

Bash
docker-compose stop
